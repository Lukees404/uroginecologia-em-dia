# Assets de Imagens

Este diretório contém as imagens do projeto.

## Imagens Necessárias

### Favicon
- ✅ `../favicon.ico` - Criado (SVG placeholder - recomenda-se substituir por ICO real 32x32)
- ✅ `../favicon.svg` - Criado (favicon moderno para browsers atuais)

### Open Graph Images (Compartilhamento Social)
Crie estas imagens para melhorar o compartilhamento nas redes sociais:

- ⚠️ `og-home.jpg` - 1200x630px - Para a página inicial
- ⚠️ `og-sobre.jpg` - 1200x630px - Para a página "Sobre Nós"
- ⚠️ `og-protocolos.jpg` - 1200x630px - Para página de protocolos
- ⚠️ `og-artigos.jpg` - 1200x630px - Para página de artigos
- ⚠️ `og-eventos.jpg` - 1200x630px - Para página de eventos
- ⚠️ `og-default.jpg` - 1200x630px - Imagem padrão

## Recomendações

### Favicon
- **Tamanho**: 32x32px ou 16x16px
- **Formato**: .ico (melhor compatibilidade) ou .svg (moderno)
- **Design**: Logo simples, legível em tamanho pequeno
- **Cores**: Use as cores do brand (azul #1e40af)

### Open Graph
- **Tamanho**: 1200x630px (proporção 1.91:1)
- **Formato**: JPG ou PNG
- **Peso**: Máximo 300KB
- **Conteúdo**:
  - Logo do site
  - Título da seção
  - Background com cores do brand
  - Evite texto muito pequeno

## Ferramentas Recomendadas

- **Favicon Generator**: https://realfavicongenerator.net/
- **Open Graph Image Maker**: https://www.canva.com/ (template 1200x630)
- **Image Optimizer**: https://tinypng.com/
